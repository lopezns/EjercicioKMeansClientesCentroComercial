Minería de Datos

Segmentación de clientes usando K-Means Clustering

Esta aplicación web permite analizar clientes de un centro comercial y agruparlos automaticamente según su ingreso anual y su puntaje de gasto.

Ejecutar modelo
Dataset Mall Customers 200 registros Variables: Ingreso anual y puntaje de gasto Link del Dataset: Kaggle

Descripción del problema

Un centro comercial necesita identificar grupos de clientes con comportamientos similares para tomar mejores decisiones de mercadeo, fidelización y promociones. El algoritmo K-Means permite descubrir segmentos sin usar etiquetas previas.

Objetivo

Implementar una aplicacion en Flask que cargue un dataset, prepare los datos, aplique K-Means y visualice clusters, centroides y método del codo, para llevar a cabo el análisis de segmentación de clientes.

Resultado esperado

Una gráfica de dispersión con los clientes agrupados, una tabla con la etiqueta asignada a cada registro y una interpretación de los patrones encontrados.
